RAPI SANG CEO MERCEDES tai

saya lahir di karawang kosambi 12 juni 1997 saya memiliki ibu bernama suryati dan ayah bernama sumarno.disini saya akan menceritakan kisah hidup saya hingga bisa menjadi orang sukses.

saya dari kecil belajar bisnis dari ayah saya yang dulu berbisnis jualan sepatu di daerah kosambi.saya dari kecil juga sudah di ajari cara mengelola bisnis sepatu bapak saya.pada suatu hari bapak saya mengalami kerugian karna menurun nya pembeli hingga bapak saya memutuskan untuk berhenti bisnis sepatu

karna hal itu membuat saya menjadi termotivasi untuk membantu ekonomi keluarga saya.saya belajar dari pengusaha pengusaha besar bagaimana dia membangun bisnis besar sehingga mereka sukse.saya belajar dan belajar.

suatu hari saya mengalami kesuliatan karna bapak saya tidak bisa membayar uang untuk melanjutkan sekolah saya.namun tidak di sangka sangka saya mendapatkan beasiswa dari smk hingga tamat kuliah.di situ membuat saya yakin kalo saya sungguh sungguh saya bisa menjadi orang sukses.

saya kuliah masuk jurusan tata usaha di sana saya banyak belajar bagaimana membangun usaha dari awal hingga akhir di situ saya banyak belajar dan saya kepikiran kalo saaya membuka usaha cuci mobil.singkat cerita usaha cuci mobil yang saya buka laku keras sehingga saya ingin mengubah usaha saya menjadi pembuat mobil mercedes.dan tidak di sangka usaha itu pun laku keras.dan akhir nya saya bisa membanggakan kedua orang tua saya dan adik saya
